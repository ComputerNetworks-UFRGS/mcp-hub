# TC004 — Orders: Excessive CPU Usage

| Field            | Value                          |
|------------------|--------------------------------|
| **Fault type**   | Code bug                       |
| **Service**      | `orders`                     |
| **Image tag**    | `minimal-boutique-orders:tc004`|
| **Introduced by**| Source code change             |

---

## Fault description

Whenever an user requests GET /orders, an artifical load is generated for the CPU.

---

## Setup
```bash 
kubectl set image deployment/orders orders=ghcr.io/computernetworks-ufrgs/minimal-boutique-orders:tc004 -n minimal-boutique
```

---

## Teardown

```bash
kubectl set image deployment/orders orders=ghcr.io/computernetworks-ufrgs/minimal-boutique-orders -n minimal-boutique
```

## Example prompts
