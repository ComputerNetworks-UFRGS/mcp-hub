# TC006 — Payment: Trying to pay throws an exception

| Field            | Value                          |
|------------------|--------------------------------|
| **Fault type**   | Code bug                       |
| **Service**      | `payment`                     |
| **Image tag**    | `minimal-boutique-payment:tc006`|
| **Introduced by**| Source code change             |

---

## Fault description

POST /payment/charge causes the payment service to throw an exception, causing errors and generating logs.

---

## Setup
```bash 
kubectl set image deployment/payment payment=ghcr.io/computernetworks-ufrgs/minimal-boutique-payment:tc006 -n minimal-boutique
```

---

## Teardown

```bash
kubectl set image deployment/payment payment=ghcr.io/computernetworks-ufrgs/minimal-boutique-payment -n minimal-boutique
```

---

## Example prompts

- The system doesn't allow me to pay for my purchase.
