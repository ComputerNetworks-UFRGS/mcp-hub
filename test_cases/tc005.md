# TC005 — Orders: Create Orders Returns 400

| Field            | Value                          |
|------------------|--------------------------------|
| **Fault type**   | Code bug                       |
| **Service**      | `orders`                     |
| **Image tag**    | `minimal-boutique-orders:tc005`|
| **Introduced by**| Source code change             |

---

## Fault description

Whenever POST /orders is requested, it returns a 400 error. 
POST /orders is used to checkout the items from the cart

---

## Setup
```bash 
kubectl set image deployment/orders orders=ghcr.io/computernetworks-ufrgs/minimal-boutique-orders:tc005 -n minimal-boutique
```

## Teardown
